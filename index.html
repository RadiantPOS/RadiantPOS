Welcome! Here are some things to note


Here is my personal project of many months; a Culver's POS training simulator! This system is based off many Culver's now old POS systems, many are upgrading to a newer updated one. This version includes BASIC features of the POS system. This is not a full fledged recreation. Most of the sidebar buttons do not work, and niether do the scroll buttons, they are there for show. It does however have the following key features:


  • 1. It includes most menu items, plus regular and small value baskets, dinners, etc with correct pricing and upcharges (for Michigan)
  • 2. It has MOST screens, such as sauces, special instructions, dessert mixins, etc.
  • 3. It properly determines the price upcharges between regular and small baskets, and kids meals.
  • 4. The "Cancel Item" button DOES work

Keep in mind:

  • 1. This does not include quite a few submenus, such as dinner options, a few submenus for custard, etc.
  • 2. This was created in the process of 7 months while juggling work and school, so it absolutely isn't perfect. There are bugs here and there, and some of it is not accurate to the real version.
  • 3. I finished this after I quit Culver's, so I wasn't able to get all the details right.
  • 4. While this is a great start to getting familiar with the register, it is not exact. This includes basic functionality that'll be a good refresher for current employees, and a decent first look for new employees.
  • 5. This simulator does not look good on very small screens. There are a few visual bugs when the screen size gets smaller.
